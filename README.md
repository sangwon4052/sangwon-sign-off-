# 🎓 광주대학교 총동문회 관리시스템

> **Gwangju University Alumni Association Management System**  
> GitHub Pages로 즉시 배포 가능한 순수 HTML 웹 애플리케이션

---

## 🌐 바로 사용하기 (GitHub Pages 배포)

### 1단계 — GitHub 저장소 만들기
1. [github.com](https://github.com) 접속 → 로그인
2. 우측 상단 `+` 버튼 → **New repository** 클릭
3. Repository name: `gwangju-alumni` (원하는 이름 가능)
4. **Public** 선택 (Pages 무료 배포용)
5. **Create repository** 클릭

### 2단계 — 파일 업로드
1. 저장소 페이지에서 **Add file → Upload files** 클릭
2. `index.html` 파일을 드래그하거나 선택
3. 아래 Commit changes 버튼 클릭

### 3단계 — GitHub Pages 활성화
1. 저장소 상단 **Settings** 탭 클릭
2. 좌측 메뉴에서 **Pages** 클릭
3. Source: **Deploy from a branch**
4. Branch: **main** / **/ (root)** 선택 후 **Save**
5. 잠시 후 상단에 주소 표시:  
   👉 `https://[내GitHub아이디].github.io/gwangju-alumni/`

> ⏱️ 배포까지 약 **1~3분** 소요됩니다.

---

## 📁 파일 구조

```
gwangju-alumni/
├── index.html      ← 메인 파일 (이 파일 하나로 전체 동작)
└── README.md       ← 이 설명 파일
```

> **단 1개의 파일(index.html)** 으로 모든 기능이 동작합니다.  
> 이미지, 스타일, 스크립트가 모두 포함되어 있습니다.

## 💾 데이터 저장 방식

- 모든 데이터는 **브라우저 localStorage**에 저장됩니다
